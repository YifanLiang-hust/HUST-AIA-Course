## 📖 项目简介

本项目是华中科技大学人工智能与自动化学院（HUST AIA）的课程学习资料汇总，旨在为学院师生提供一个集中的学习资源平台。项目收录了本人在本科和研究生阶段的重要课程资料，包括但不限于课程论文、项目代码。

### 🎯 项目目标
- 📚 整理和分享优质的课程学习资料
- 🤝 促进学院内的学术交流与合作
- 💡 为后续学习者提供参考和指导
- 🌟 构建开放共享的学习资源库
